#  Prerak AI | AI-Powered DevRel Ambassador Platform

**Prerak AI** is a centralized, gamified Campus Ambassador management system that completely automates technical onboarding and task tracking. 

##  The Problem We Solve
Organizations struggle to manage technical campus ambassadors. Tracking is fragmented across WhatsApp and spreadsheets, and evaluating a student's technical capability takes too long. 

##  Our Solution
We bridge the gap by combining an automated **GitHub Profile Analyzer** with a **Gamified Task Dashboard**.
1. **Instant Evaluation:** Students apply via their GitHub ID. Our Python-based AI engine analyzes their code quality, commits, and repos in under 2 minutes, giving them a "Dev Score" and showing them exactly what recruiters notice first.
2. **Automated Workflows:** Managers assign tasks (e.g., Open Source PRs, tech blogs). Our system auto-verifies proof-of-work using NLP.
3. **Gamified Retention:** Real-time leaderboards, badges, and streak mechanics to keep community leads and ambassadors highly engaged.

##  Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js, MongoDB
* **AI Engine:** Python (FastAPI), GitHub REST API, LLM Integration (RAG-based evaluation)
